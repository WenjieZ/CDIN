Name explanation
----
- meta  = metadata
- alt   = alternative
- ol    = ordered list
- ul    = unordered list
- li    = list item
- a     = anchor
- img   = image
- tr    = table row
- th    = table head
- td    = table data

Suggestion
----
The choice of text editor has nothing to do with your capcity of coding. Don't waste too much time on the choice of editor.
